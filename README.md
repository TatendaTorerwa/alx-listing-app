# ALX Listing App

## Description

ALX Listing App is a project aimed at creating the foundational structure for an Airbnb clone. This milestone sets up the Next.js app with TypeScript, ESLint, and TailwindCSS to build a dynamic property listing page.

## Project Structure

- `components/`: Contains reusable components like `Card.tsx` and `Button.tsx`.
- `interfaces/`: Holds TypeScript interfaces like `CardProps` and `ButtonProps`.
- `constants/`: Stores reusable constants such as API URLs and UI text.
- `public/assets/`: Contains all images and SVGs used across the project.
